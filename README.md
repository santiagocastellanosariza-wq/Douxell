# Douxell

Página web inicial para la marca Douxell.

## Identidad visual

- Verde mata como color principal.
- Blanco, dorado suave, beige y cacao como colores de apoyo.
- Estilo premium, artesanal y natural.

## Productos visibles por ahora

- Cacao natural Douxell 250 g.
- Chips de chocolate Douxell 250 g.

No se incluyeron granola ni quinoa, según la indicación actual de marca.

## Archivos

- `index.html`: estructura de la página.
- `style.css`: diseño visual y responsive.
- `script.js`: menú móvil, carrito visual, popup y formularios.
